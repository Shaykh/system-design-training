# Diagrammes Draw.io — ShopFlow

Les fichiers `.drawio` de ce dossier sont des **emplacements** pour vos exports visuels. Le contenu de référence est en Mermaid dans [README.md](README.md).

---

## Procédure

1. Ouvrir [diagrams.net](https://app.diagrams.net) (Draw.io)
2. Créer un diagramme selon le niveau C4
3. S'inspirer des diagrammes Mermaid du README
4. Enregistrer sous le nom correspondant :
   - `context-diagram.drawio`
   - `container-diagram.drawio`
   - `component-diagram.drawio`
5. Exporter en PNG pour slides / DAT

## Alternative sans Draw.io

Les diagrammes Mermaid du README sont **suffisants** pour la formation et la soutenance. Draw.io est recommandé pour un livrable entreprise plus soigné.
